When working with designer spec, sometimes it’s hard to measure dimensions by looking at the screen. Pixel Ruler gives you perfect tool to get dimensions. *\[Get it from* [*Google Play*](https://play.google.com/store/apps/details?id=com.bidyut.app.pixelruler)*\]*

### Show Java

Sometimes I need to validate some values that are in built APK after release. For example, debug mode is NOT on, or version number is correct from CI build and so on. “[Show Java](https://play.google.com/store/apps/details?id=com.njlabs.showjava)” is a perfect app to decompile app with surprisingly accurate reconstruction of both Java source and XML resource files. Previously I used to use [Dexplorer](https://play.google.com/store/apps/details?id=com.dexplorer), but now I am not looking back. Bought the pro version to support developer. \[Get it from [Google Play](https://play.google.com/store/apps/details?id=com.njlabs.showjava)\]
